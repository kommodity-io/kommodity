--- Uninstall ---

`helm uninstall {{ .Release.Name }} -n {{ .Release.Name }}` will delete the
"{{ .Release.Name }}" Namespace and EVERYTHING inside it, including:

  - CAPI-owned secrets (kubeconfig, bootstrap data, talos config)
  - KMS Secrets (`talos-kms-<uuid>`) — disk-encryption keys for every node
  - Provider credentials Secret if you created it in this namespace
  - Anything else you placed in the namespace manually

If you need to keep any of those, copy them to another namespace before
uninstalling.
